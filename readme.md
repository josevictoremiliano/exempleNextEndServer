
# Exemplo de execução de server js e next js



## Rodando localmente

### Inície o projeto server

Entre no diretório do server

```bash
  cd server
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```

### Inície o projeto web

Entre no diretório do server

```bash
  cd web
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
